# Getting Started

Quick guide to understanding and deploying this infrastructure.

## What You'll Learn

- Deploy production-grade infrastructure
- Docker container orchestration
- Reverse proxy configuration
- SSL/TLS management
- Workflow automation
- Infrastructure as Code

## Prerequisites

**Accounts Needed:**
1. Hetzner Cloud (~€7/month)
2. Domain name (~€10/year)
3. Cloudflare (free)

**Skills Needed:**
- Basic Linux command line
- Understanding of DNS
- SSH access

## Quick Start (2-3 hours)

### 1. Infrastructure (30 min)
- Create Hetzner server
- Install Coolify
- Configure Cloudflare DNS

### 2. Services (45 min)
- Deploy n8n workflow automation
- Deploy Open WebUI for AI
- Configure domains and SSL

### 3. Secure (30 min)
- Set up automated backups
- Configure monitoring
- Test disaster recovery
